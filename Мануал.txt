Для того чтобы запустить сборку, необходимо
Поставить программу NodeJS,
Далее необходимо поставить консоль git bash или cmder mini
cmder mini приложу в файлы. Один раз нужно запустить консоль, затем можно запускать прям  папку проекта.
Затем в косоли необходимо выполнить две команды
npm install
gulp build
